BLUG Meeting (July 2015) - Red Programming Language

Jul 14 (Tue)
By Admin

In misc.

2015-07-14 16:10:00

Nenad Rakocevic will tell us about his programming language "red".

Red is a homoiconic language similar to Rebol. It is a "full stack programming language" which can be used for extremely high-level programming (DSLs and GUIs) as well as low-level programming (operating systems and device drivers).

We will also discuss what our new website should be like. Welcome every one join this discussion.
